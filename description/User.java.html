This is one of the two beans I created for this project. This bean is used to display information about a user. Here is a breakdown of its member variables: identifier: Random identifier for the user. username: Chosen at registration. password: Chosen at registration. email: Chosen at registration but alterable later. cart: An arraylist of Item objects, each item is considered "in" the user's cart. The rest are trivial and immediately evident what their purpose is... When a user is created it first generates a random identifier for the User and then assigns the username, password, and email, as well as initializes an empty cart. Everything else is initialized to blank because it isnt required information until checkout time.
