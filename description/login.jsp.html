This is the entry view of the web app. If you have no session or action, you will be routed here. The nav buttons are still displayed here, but they are all disabled until you are logged in, as they are in the other non-logged-in views. But that doesn't matter anyway, without a user identifier you will be routed to an error page anyway. Also notice that I rely heavily on bootstrap's grid system for my view layout, this is a common theme throughout the site.
